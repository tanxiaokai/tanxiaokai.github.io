发表于 2016-12-16 | 分类于 其它 ， blog系统

创建新文章

新文章创建非常简单，是需要输入下面的命令或者直接在 source/_posts 目录下创建.md文件即可。 

 hexo new "文章标题"

这个命令会自动在source/_posts下生成一个文章标题.md的文件。用文本编辑器打开这个文章可以看到，以下代码。文件的开头是属性，采用统一的yaml格式，用三条短横线分隔。下面是文章正文。新建、删除或修改文章后，不需要重启hexo server，刷新一下即可预览。


---
title: 文章标题                   //文章标题
date: 2016-12-16 10:46:21        //写文章日期
tags:                            //Tag标签，这里是同时使用多个标签的写法
 - Tag1
 - Tag2 
categories:                      //文章分类，这里用多级分类的写法
 - 一级分类
 - 二级分类
 - 三级分类

---
文章正文
文章可以拥有如下属性
属性名 描述 默认值
layout Layout post或page
title 文章的标题
date 创建日期 文件的创建日期
updated 修改日期 文件的修改日期
comments 是否开启评论 true
tags 标签
categories 分类
permalink url中的名字 文件名

写文章内容

Hexo是使用Markdown编辑文章的，这个语言非常简洁书写起来不会像HTML那样有一大堆标签代码。虽然Markdown语言能兼容Html，但是还是极力推荐使用Markdown语言编写。

  1. Markdown语言

    Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯和亚伦·斯沃茨。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML文档”。 ——维基百科
    他有一些具体的语法，这里我就不多说了，可以直接查看Markdown语法［http://www.appinn.com/markdown/#p]

  2. Markdown编辑工具

    我使用的Mac OS X，如果你也是我强烈建议你用 Mou 这款免费且十分好用的 Markdown 编辑器，它支持实时预览，既左边是你编辑 Markdown 语言，右边会实时的生成预览效果。不仅如此，Mou 还有一些有趣的偏好设置（Preference），例如主题（Themes）与样式（CSS），它们可以配置出定制化的文本编辑效果与导出效果，如果你对自带的主题与样式不满意还可以到 GitHub 上搜索其它爱好者为 Mou 编写的更多主题样式，导入的方式可以在偏好设置的 Themes 或 CSS 选项中 选择 reload。

    Mou下载地址 [http://25.io/mou/]

    Windows，有两款还算不错，一款叫做 MarkdownPad ，另一款叫做 MarkPad。

    需要特别说明一下，这个编辑器如果要强制换行使用 Control + Return插入换行符。下面有很多快捷方式，也许你不记得Markdown的一些标示，可以使用这些快捷件帮助你。

插入本地图片

使用markdown写文章，插入图片的格式为

在hexo/source目录下新建一个img文件夹，将图片放入该文件夹下，插入图片时链接即为
![图片名称](/img/图片名称)
