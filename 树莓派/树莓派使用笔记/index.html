树莓派使用笔记

发表于 2018-03-16 | 分类于 树莓派

​ Raspberry Pi(中文名为“树莓派”,简写为RPi，(或者RasPi / RPI)[1] 是为学习计算机编程教育而设计，只有信用卡大小的微型电脑

​ 自问世以来，受众多计算机发烧友和创客的追捧，曾经一“派”难求。别看其外表“娇小”，内“心”却很强大，视频、音频等功能通通皆有，可谓是“麻雀虽小，五脏俱全”

一、安装系统

  • 官网 下载 Raspbian 最新版本。RASPBIAN STRETCH LITE是无桌面版本。
  • 8Gb以上tf卡。通过工具把系统镜像写入tf卡。
  • 插入树莓派启动
# Mac做镜像命令
df -h  #查看U盘盘符
diskutil unmount /dev/盘符  #分区卸载
diskutil list #查询设备号
dd bs=4m if=镜像文件.img of=写入设备号 #写入镜像

写入完成 卸载
diskutil unmountDisk /dev/disk1

Mac参考文档

二、系统配置

  • 默认账号: pi 密码: raspberry
  • 开启SSH
sudo raspi-config
#选择Interfacing Options Configure -> SSH -> ENABLE
  • 调整音量
sudo alsamixer

#设置声卡通过耳机接口输出
sudo amixer cset numid=3 1

#测试播放声音
sudo aplay /usr/share/sounds/alsa/Front_Center.wav
  • 升级apt-get
sudo apt-get update
  • 开启root
#用pi用户，执行任何命令都有加sodu很蛋疼
sudo passwd root
sudo passwd --unlock root

#登录root
su root
  • 更换阿里云源
#sudo apt-get install -y vim
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak   #备份为 sources.list.bak
sudo vi /etc/apt/sources.list

追加数据: 
deb http://mirrors.aliyun.com/raspbian/raspbian/ wheezy main non-free contrib
deb-src http://mirrors.aliyun.com/raspbian/raspbian/ wheezy main non-free contrib

sudo apt-get update && apt-get upgrade -y       #更新系统软件 并 更新已安装的包
  • 安装必要软件
sudo apt-get install -y vim git telnet sshpass nodejs npm supervisor

#sudo apt-get install -y vim
#sudo apt-get install -y git
#sudo apt-get -y install telnet
#sudo apt-get -y install sshpass
#sudo apt-get install -y nodejs
#sudo apt-get install -y npm
#sudo apt-get install -y supervisor
  • 设置静态IP
ifconfig  #获取网卡名
sudo vim /etc/dhcpcd.conf
#添加
interface 网卡名
static ip_address=10.1.1.149/24
static routers=10.1.1.1
static domain_name_servers=10.1.1.1 8.8.8.8

ifconfig eth0 mtu 1200

sudo reboot #重启

#生成ssh key
ssh-keygen -t rsa -C "pi-test"

三、安装软件

  • JDK 下载arm版本
 #创建目录
mkdir /opt/install
mkdir /opt/install/script

mkdir /opt/application
mkdir /opt/application/yard_service
mkdir /opt/application/eis

mkdir /opt/backup 
mkdir /opt/deploy
mkdir /opt/logs
mkdir /opt/operation

mkdir /data
mkdir /data/m2lib

cd /opt/

sudo wget --no-cookies --no-check-certificate --header "Cookie: gpw_e24=http%3A%2F%2Fwww.oracle.com%2F; oraclelicense=accept-securebackup-cookie" "http://download.oracle.com/otn-pub/java/jdk/8u161-b12/2f38c3b165be4555a1fa6e98c45e0808/jdk-8u161-linux-arm32-vfp-hflt.tar.gz?AuthParam=1521008208_37391eb2ef3c337e4a4b1dab7507a457"

tar xzf jdk*
cd /opt/jdk1.8.*/

update-alternatives --install /usr/bin/jar jar /opt/jdk*/bin/jar 1
update-alternatives --install /usr/bin/javac javac /opt/jdk*/bin/javac 1
update-alternatives --set jar /opt/jdk*/bin/jar
update-alternatives --set javac /opt/jdk*/bin/javac
update-alternatives --install /usr/bin/java java /opt/jdk*/bin/java 1

update-alternatives --config java

#添加变量
vi /etc/environment

#添加以下内容：
export JAVA_HOME=/opt/jdk1.8.0_131
export JRE_HOME=/opt/jdk1.8.0_131/jre
export MAVEN_HOME=/usr/local/maven3

. /etc/environment 执行

java -version  #查看java版本
  • 加载讯飞语音包
ldd  /opt/deploy/bm_radio/libmsc32.so
ldd  /opt/deploy/bm_radio/libmsc64.so

cp /opt/deploy/bm_radio/bm_radio/*.so /lib/
