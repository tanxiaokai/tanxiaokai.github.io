Redis安装

发表于 2016-12-20 | 分类于 开发 ， Redis

Mac/Linux 下安装

  1. 官网http://redis.io/ 下载最新的稳定版本,这里是3.2.6

    安装wget 

     sudo brew install wget

    下载安装包

     sudo wget http://download.redis.io/releases/redis-3.2.6.tar.gz

  2. 解压安装包

     sudo mv redis-3.2.6.tar.gz /usr/local/
 cd /usr/local/
 sudo tar -zxf redis-3.2.6.tar.gz
 cd redis-3.2.6

  3. 安装

     sudo make test #测试编译
 sudo make install #安装

  4. 服务器运行测试

    cd /usr/local/redis-3.2.6/
sudo ./runtest

    成功结果如图

  5. 启动服务

     sudo /usr/local/bin/redis-server

服务配置

  1. 建立相关目录

     mkdir /usr/local/redis
 mkdir  /usr/local/redis/bin
 mkdir  /usr/local/redis/etc
 mkdir  /usr/local/redis/db
 mkdir /usr/local/redis/log

  2. 拷贝文件

     cp /usr/local/bin/redis-* /usr/local/redis/bin/
 cp /usr/local/redis-3.2.6/src/mkreleasehdr.sh /usr/local/redis/bin/
 cp /usr/local/redis-3.2.6/redis.conf /usr/local/redis/etc/
 cp /usr/local/redis-3.2.6/utils/redis_init_script /usr/local/redis/bin/

  3. 修改 redis.conf

    配置详解 redis.conf配置详解

     vi /usr/local/redis/etc/redis.conf 
 #修改为守护模式
 daemonize yes
 #设置进程锁文件
 pidfile /usr/local/redis/redis.pid
 #端口
 port 6379
 #客户端超时时间
 timeout 300
 #日志级别
 loglevel debug
 #日志文件位置
 logfile /usr/local/redis/log/log-redis.log
 #指定本地数据库文件名
 dbfilename dump.rdb
 #指定本地数据库路径
 dir /usr/local/redis/db/

  4. 启动服务 

     /usr/local/redis/bin/redis-server /usr/local/redis/etc/redis.conf
 tail -n 100 /usr/local/redis/log/log-redis.log

  5. 检查服务

    检测后台进程是否存在

     ps -ef |grep redis

    检测6379端口是否在监听

     netstat -lntp | grep 6379

    使用redis-cli客户端检测连接是否正常

     /usr/local/redis/bin/redis-cli
 127.0.0.1:6379> keys *
 (empty list or set)
 127.0.0.1:6379> set key "hello world"
 OK
 127.0.0.1:6379> get key
 "hello world"

  6. 停止服务

     /usr/local/redis/bin/redis-cli shutdown

  7. 设置开机自启动

    推荐在生产环境中使用启动脚本方式启动redis服务。启动脚本 redis_init_script 位于位于Redis的 /usr/local/redis-3.2.6/utils 目录下。

     ln -s /usr/local/redis/bin/redis_init_script /etc/init.d/redisd
 ln -s /usr/local/redis/etc/redis.conf /etc/redis/6379.conf

    在启动脚本开头添加如下两行注释以修改其运行级别：

     #!/bin/sh
 # chkconfig:   2345 90 10
 # description:  Redis is a persistent key-value database
 #

    设置为开机自启动服务器

     chkconfig redisd on

    打开服务

     service redisd start

    关闭服务

     service redisd stop

安装出现的问题和解决方案

  1. 错误

     [root@BONCDT redis-3.2.6]# make test
 cd src && make test
 make[1]: 进入目录“/home/dwinst/redis/redis-3.2.6/src”
 You need tcl 8.5 or newer in order to run the Redis test
 make[1]: *** [test] 错误 1
 make[1]: 离开目录“/home/dwinst/redis/redis-3.2.6/src”
 make: *** [test] 错误 2

    解决, 安装tcl

     yum install tcl

    或

     wget http://downloads.sourceforge.net/tcl/tcl8.6.1-src.tar.gz  
 sudo tar xzvf tcl8.6.1-src.tar.gz  -C /usr/local/  
 cd  /usr/local/tcl8.6.1/unix/  
 sudo ./configure  
 sudo make  
 sudo make install
