Jenkins安装与配置

发表于 2017-02-16 | 分类于 维护 ， Jenkins

Jenkins介绍

Jenkins是基于Java开发的一种持续集成工具，用于监控持续重复的工作，功能包括：

  1. 持续的软件版本发布/测试项目。
  2. 监控外部调用执行的工作。

安装步骤

1.下载

从官网下载jenkins的war包，地址https://jenkins.io/

wget https://mirrors.tuna.tsinghua.edu.cn/jenkins/war-stable/2.89.4/jenkins.war

2.启动

java -jar jenkins.war -Dfile.encoding=utf-8 --httpPort=8888 &

3.首次访问

#查看初始化密码
cat /root/.jenkins/secrets/initialAdminPassword

#访问地址，输入密码登录, 
http://127.0.0.1:8888
